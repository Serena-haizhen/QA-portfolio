# Web Basics: Cookie, Auth, Query Params and JSON

## Cookie
A cookie is a small piece of information stored in the browser.
It can help the system remember login status or user settings.

## Auth
Auth is used to verify who the user is.

## Authentication
Authentication means identifying who the user is.

## Authorization
Authorization means what the user is allowed to do.

## Query Params
Query params are extra parameters added to the end of a URL.
They are often used for search, filtering, sorting, or pagination.

## JSON
JSON is a common data format that stores information as key-value pairs.

## Key Reminder
- Authentication = who you are
- Authorization = what you can do
