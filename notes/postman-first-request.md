# Postman First Request

## What I did
Today I used Postman to send my first GET request.

## Request
GET https://jsonplaceholder.typicode.com/posts/1

## Result
- Method: GET
- Status: 200 OK
- Response body was returned successfully

## What I learned
- Postman can send API requests without using a web page
- GET is used to get data
- The API returned JSON data
- Status 200 means the request was successful
