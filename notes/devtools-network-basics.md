# DevTools Network Basics

## What I learned
Today I opened the browser DevTools and used the Network tab to see real requests.

## Key Fields
- Name: the request name
- Status: the result status code
- Type: the kind of request

## First Observations
- document = the page itself
- status 200 = the request was successful

## Why it matters
The Network tab helps testers see what happens behind the page, not only what is shown on the screen.
