# HTTP Basics

## What is HTTP
HTTP is a way for the browser and server to communicate.

## Request
A request is sent from the browser to the server to ask for data or send information.

## Response
A response is the result returned by the server after receiving a request.

## Methods
### GET
GET is used to get data.

### POST
POST is used to send or submit data.

## Common Status Codes
### 200
The request was successful.

### 400
The request was bad or invalid.

### 401
The user is not authenticated.

### 403
The user is authenticated but does not have permission.

### 404
The requested page or resource was not found.

### 500
The server had an internal error.
