# Basic GET Requests

## 1. Health Check
- Method: GET
- URL: https://restful-booker.herokuapp.com/ping
- Result: Response received successfully
- Status: 201 Created

## 2. Get All Booking IDs
- Method: GET
- URL: https://restful-booker.herokuapp.com/booking
- Result: Response body returned successfully
- Status: 200 OK
- Response Type: List

## 3. Get Single Booking
- Method: GET
- URL: https://restful-booker.herokuapp.com/booking/1
- Result: Response body returned successfully
- Status: 200 OK
- Response Type: Booking details
