# Test Summary

## Scope
This test pack covers the Login and Dropdown modules from The Internet demo site.

## What I tested
- Basic positive and negative login scenarios
- Empty input validation for login
- Initial dropdown behavior, including default state, expanding the dropdown, and selecting an option

## Key Findings
- Valid credentials may fail to log in
- Login may succeed when the password field is empty
- Dropdown basic behavior was observed and covered with initial test cases
