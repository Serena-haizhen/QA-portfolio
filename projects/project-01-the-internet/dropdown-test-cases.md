# Dropdown Test Cases

## DROPDOWN-001

**Title:** Verify that the dropdown is displayed in the default state

**Preconditions:**
- The dropdown page is available

**Steps:**
1. Open the dropdown page

**Expected Result:**
1. The dropdown is visible
2. The dropdown is collapsed by default
3. The default text or value is displayed

**Actual Result:**
1. The dropdown is visible
2. The dropdown is collapsed by default
3. The default text or value is displayed

**Status:** Pass

---

## DROPDOWN-002

**Title:** User can expand the dropdown list

**Preconditions:**
- The dropdown page is available

**Steps:**
1. Open the dropdown page
2. Click the dropdown

**Expected Result:**
1. The dropdown list expands
2. The available options are displayed

**Actual Result:**
1. The dropdown list expanded successfully
2. The available options were displayed

**Status:** Pass

---

## DROPDOWN-003

**Title:** User can select Option 1

**Preconditions:**
- The dropdown page is available

**Steps:**
1. Open the dropdown page
2. Click the dropdown
3. Select Option 1

**Expected Result:**
1. Option 1 is selected
2. The dropdown displays Option 1 as the selected value

**Actual Result:**
1. Option 1 was selected successfully
2. The dropdown displayed Option 1 as the selected value

**Status:** Pass
